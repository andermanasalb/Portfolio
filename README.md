# Ander Mañas - Portfolio

High-performance, interactive personal portfolio designed to showcase expertise in Full-Stack Development, Aerospace Engineering, and AI-driven solutions.

## 🚀 Key Features

- **Highly Interactive Interface:** Advanced use of fluid animations, micro-interactions, and visual effects.
- **Modern & Responsive Design:** Premium dark aesthetic (Vercel/Linear style) featuring glassmorphism, dynamic blurs, and perfect adaptation across all devices.
- **3D & Graphic Elements:** Integration of interactive 3D components for enhanced user immersion.
- **Dynamic Sections:**
  - *Hero:* Main interface featuring an integrated interactive chat simulator.
  - *About:* Interactive information cards with spatial hover effects, sliders (e.g., chess minigame link), and dynamic radar maps.
- **Integrated Contact System:** Fully functional direct contact form with real-time user notifications via Sonner.
- **Optimized Performance:** Ultra-fast loading times leveraging Next.js Server-Side Rendering (SSR) and built-in optimizations.

## 💻 Tech Stack

This project is built using the latest web ecosystem technologies to ensure robustness, maintainability, and an AAA user experience:

- **Core:** React 19, Next.js 15 (App Router)
- **Language:** TypeScript
- **UI & Styling:** Tailwind CSS 4, Radix UI, Lucide React (Icons)
- **Animations:** Framer Motion, GSAP (GreenSock)
- **3D Graphics:** Spline
- **Email / Notification Management:** Resend (email delivery), Sonner (Toasts)

## 📂 Project Structure

- **/src/app:** Base routes and application pages using the Next.js App Router.
- **/src/components:** Reusable UI components (like the `about.tsx` section, hero, cards).
- **/src/lib:** Utility functions and helpers (e.g., Tailwind class merger).
- **/public:** Static assets, textures, profile pictures, and maps.
